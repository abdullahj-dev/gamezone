# GameZone 🎮

A multi-game web platform built with Next.js.
Multiple games in one place, fully responsive.
26 active games including multiplayer options.

## Tech Stack
- Next.js
- JavaScript
- HTML & CSS

## Live Demo
🔗 [Play Now](https://gamezone-kappa-rosy.vercel.app/)

## Games Included
Flappy Bird, Snake, Mnemonics, Fleet, Shadow, Depths, 
Art-board, Spin-Blades, Battle-Ship, 2048, Mine-Sweeper and more...
